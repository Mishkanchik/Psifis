# Псифіс - Платформа пошуку психологів

Сучасна платформа для пошуку кваліфікованих психологів та психотерапевтів, створена з використанням React Vite та TailwindCSS.

## 🚀 Функціонал

- **Пошук психологів** за категоріями (особиста, парна, дитяча терапія, психіатрія)
- **Фільтрація** за спеціалізацією та пошук за іменем
- **Каталог спеціалістів** з рейтингами, відгуками та цінами
- **Популярні теми** для швидкого пошуку за проблемою
- **Адаптивний дизайн** для мобільних та десктопних пристроїв
- **Сучасний UI** з градієнтами та анімаціями

## 🛠️ Технології

- **React 18** - UI бібліотека
- **Vite** - швидкий бандлер та сервер розробки
- **TailwindCSS** - утилітарний CSS фреймворк
- **Lucide React** - іконки

## 📦 Встановлення

1. Встановіть залежності:
```bash
npm install
```

2. Запустіть сервер розробки:
```bash
npm run dev
```

3. Збірка для продакшену:
```bash
npm run build
```

## 🎨 Дизайн

Проект використовує унікальну кольорову схему з градієнтами:
- Primary: синій відтінок (sky blue)
- Accent: фіолетовий відтінок
- Фон: градієнт від світло-сірого до блакитного
